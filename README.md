# Photobackup Project

## Ultima Modifica: Implementazione Iniziale API Google Drive

In questa versione, ho aggiunto il supporto di base per l'API Google Drive. Attualmente, il programma è in grado di leggere i file presenti su Google Drive.

### Modifiche apportate:
- Aggiunta dipendenza per l'API Google Drive
- Implementato il recupero e la lettura dei file da Google Drive
- Aggiunta dell'opzione per selezionare e caricare manualmente i file su Google Drive

### Miglioramenti futuri previsti:
- Implementazione di nuove funzionalità di scrittura e caricamento su Google Drive
 

Queste modifiche rendono il programma più versatile e consentono all'utente di interagire direttamente con i file da caricare su Google Drive.
